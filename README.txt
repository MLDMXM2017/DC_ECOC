Classifier.py：包含主要的ECOC分类器，每一个分类器作为一个类。该文件是最常使用的文件。
Classifier_demo.py：中包含了一个使用OVO_ECOC分类器进行训练、预测和验证的demo。
Criterion.py：包含了聚类和划分的几种评价方法，主要在D_ECOC、AGG_ECOC和CL_ECOC中使用。
Distance.py：中包含了计算距离的几种方法。
Matrix_tool.py：中包含了几个几个用于处理数据和矩阵的辅助方法。
read.py：定义了读取UCI数据集合Microarray数据集的两个方法。也是一个常用的文件。
SFFS.py：文件中实现了SFFS搜索算法。
